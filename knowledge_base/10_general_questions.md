# About This Website

## How does this chatbot work?

This chatbot is powered by a Retrieval-Augmented Generation (RAG) system. When you ask a question, it first searches my personal knowledge base for the most relevant information. That context is then provided to a Large Language Model (LLM), allowing it to generate accurate, personalized responses about my background, projects, and experience.
